# Aryamaan Roy Choudhury — Portfolio

Minimal, text-first personal portfolio built with Next.js 14, TypeScript, and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Resume

Place your resume PDF at `public/Aryamaan_Roy_Choudhury_Resume.pdf` for the Resume link to work.

## Build

```bash
npm run build
npm start
```
