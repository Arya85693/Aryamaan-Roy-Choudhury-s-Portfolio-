# Aryamaan Roy Choudhury — Portfolio

Minimal, text-first personal portfolio built with Next.js 14, TypeScript, and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Resume

The Resume button links to `public/resume.pdf`. When you update your resume, replace that file (or copy your PDF to `resume.pdf`).

## Build

```bash
npm run build
npm start
```
